# reduxライフサイクル - action

`action`は UI のイベントを表すオブジェクト。
`action`を生成する関数を`action creator`と呼ぶ。

## 使用法

`action`を`store.dispatch`に渡すとイベントが実行される。

## 実装例

```
const incrementAction = () => ({
  type: 'INCREMENT',
})

const decrementAction = () => ({
  type: 'DECREMENT',
})
```

`action`は`type`で区別される。
`action`は`payload`を持つことができる。
